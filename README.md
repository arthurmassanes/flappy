# flappy
Flappy bird in JS using Raylib 
Featured on the official [repo examples](https://github.com/RobLoach/node-raylib#examples)

## Requirements:
`cmake`

`node (>= 12)`

# How to install & run

```bash
git clone https://github.com/arthurmassanes/flappy.git
cd flappy/
npm install
npm start
# flap flap using the spacebar
```
